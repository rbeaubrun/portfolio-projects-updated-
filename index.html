Digital Entertainment Market Analysis

This project involves analyzing and visualizing 16,000 rows of global video game sales data from 1980 to 2020. Key insights include identifying top-performing game genres, platforms, and developers, with a focus on the "gaming boom" period of 2007-2010. The project provides actionable insights for investors, highlighting lucrative genres and mobile platform opportunities. This project showcases my skills in data cleaning, statistical analysis, and data visualization to uncover trends and guide strategic decision-making.
